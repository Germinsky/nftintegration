A customizable landing page for Daily Vibes Rewards platform with shortcode support and admin settings.

== Description ==

Daily Vibes Landing Page is a WordPress plugin that allows you to create a beautiful, customizable landing page for your music-based cryptocurrency rewards platform. Built specifically for artists who want to create their own on-chain economy using the Daily Vibes Rewards system.

**Features:**

* Easy shortcode integration: `[daily_vibes_landing]`
* Fully customizable through WordPress admin panel
* Edit hero section title and subtitle
* Configure call-to-action buttons
* Update contact information
* Customize color scheme
* Responsive design
* Smooth scroll animations
* Modern gradient design

**Perfect For:**

* Musicians building on-chain economies
* Artists using the Daily Vibes Rewards platform
* Web3 music projects on Base blockchain
* Token-based fan engagement systems

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/daily-vibes-landing` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to 'Daily Vibes Landing' in the admin menu to configure your settings
4. Add the shortcode `[daily_vibes_landing]` to any page or post where you want to display the landing page

== Frequently Asked Questions ==

= How do I display the landing page? =

Simply add the shortcode `[daily_vibes_landing]` to any page or post.

= Can I customize the colors? =

Yes! Go to the plugin settings page and you can customize the primary, secondary, and accent colors using the color pickers.

= Can I edit the text content? =

Absolutely! All text content including the hero title, subtitle, button text, and contact information can be edited from the plugin settings page.

= Is the landing page responsive? =

Yes, the landing page is fully responsive and works great on mobile, tablet, and desktop devices.

= Can I use this on multiple pages? =

Yes, you can use the `[daily_vibes_landing]` shortcode on multiple pages if needed.
